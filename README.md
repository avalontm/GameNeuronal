# Dino Red Neuronal

Este es un ejemplo del uso de red neuronal simple para que el dinosario pueda tomar la desision de saltar o no un obstaculo dependiendo la velocidad del juego.

# Requerido
- Visual Studio 2022
- Net6.0
- MonoGame 3.8
- Accord.Neuro
